杨辉三角中的三角形数表，是自然界和谐统一的体现。杨辉三角是二项式系数在三角形中的一种几何排列，其中蕴含着二项式系数的几个相关性质，包括二项式系数的对称性、增减性与最大值、各二项式系数的和等。

杨辉三角中不只有这些规律，它本身也包含了很多奇妙的性质。
1. 杨辉三角的前提是每行数字开头与结尾的数都为1；
2. 第n行的数字有n项，且数字之和为2n-1；
3. 在第n行中，第m个数与第n-m+1的数值相同，这便是组合数的性质之一；
4. 第n行中的第m个数可以表示为：C（n-1，m-1），也就是从n-1个元素当中选取 m-1个元素，这属于杨辉三角的基本性质之一；
5. 每个数字与上方两数之和相同，按照此规律便能够得出整个杨辉三角，即第n+1行中第i个数的数值为第n行中第i-1个数与第i个数相加之和，这便是组合数的又一大性质，用公式可以表示为：C（n+1，i）=C（n，i）+C（n，i-1）。
